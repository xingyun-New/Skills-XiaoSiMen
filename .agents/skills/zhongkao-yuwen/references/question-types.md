# 语文题型模板与示例

所有题型的标准格式、命题要求和完整示例。

## 一、字音字形（choice）

### 格式要求
- 4个选项（A/B/C/D），有且仅有1个正确答案
- 每个选项包含 2-4 个词语及其注音或字形
- 题干常见设问："下列加点字注音全部正确的一项是""下列词语中没有错别字的一项是"
- 正确答案在A/B/C/D中均匀分布

### Markdown 模板

```markdown
### 第X题（选择题·字音 | 难度：★☆☆）

**【题目】** 下列加点字注音全部正确的一项是：

A. 酝酿(niàng)  黄晕(yùn)   着落(zhuó)  咄咄逼人(duō)
B. 贮蓄(chǔ)    粗犷(kuàng)  蝉蜕(tuì)   参差不齐(cēn)
C. 侍弄(shì)    憔悴(cuì)    分歧(qí)    翻来覆去(fù)
D. 菡萏(hàn)    匿笑(nì)     攲斜(qī)    人迹罕至(hǎn)

**【答案】** D

**【知识点卡片】**
- 📚 考点：易错字音辨析
- 📖 出处：人教版七年级上册 第一单元
- 💡 关键记忆：贮蓄(zhù)不读chǔ；粗犷(guǎng)不读kuàng
- ⚠️ 易错提醒：B项「贮」应读zhù，「犷」应读guǎng，这两个是中考高频易错字音
- 🔗 关联知识：形声字读音规律——声旁不一定代表读音
```

## 二、成语/词语运用（choice）

### 格式要求
- 4个选项各含一个使用成语/词语的完整语句
- 选出使用正确/错误的一项
- 错误类型包括：望文生义、张冠李戴、褒贬误用、对象误用、语义重复

### Markdown 模板

```markdown
### 第X题（选择题·成语 | 难度：★★☆）

**【题目】** 下列句子中加点成语使用正确的一项是：

A. 这篇文章逻辑混乱，语句不通，简直不忍卒读。
B. 他为人谦和，经常对同事们耳提面命，大家都很尊敬他。
C. 景区里游人如织，大家摩肩接踵，络绎不绝。
D. 这位同学的演讲抛砖引玉，引发了在场同学们的热烈讨论。

**【答案】** C

**【知识点卡片】**
- 📚 考点：成语的正确使用
- 📖 出处：中考语文积累与运用
- 💡 关键记忆：不忍卒读——形容文章内容悲惨，不忍心读完（不是写得差）
- ⚠️ 易错提醒：A项「不忍卒读」望文生义，不能用于文章质量差；B项「耳提面命」用于长辈对晚辈，不能用于同事之间；D项「抛砖引玉」只能谦称自己
- 🔗 关联知识：谦辞与敬辞的区别
```

## 三、病句辨析（choice）

### 格式要求
- 4个选项为4个完整句子
- 选出有/无语病的一项
- 覆盖六大病句类型：语序不当、搭配不当、成分残缺或赘余、结构混乱、表意不明、不合逻辑
- 每个错误选项只含一种语病，便于学生辨析

### 示例（JSON 格式）

```json
{
  "id": 3,
  "type": "choice",
  "difficulty": 2,
  "question": "下列句子中没有语病的一项是：",
  "options": [
    "通过这次活动，使同学们更加了解了传统文化的魅力。",
    "能否保持一颗平常心，是考试正常发挥的关键。",
    "学校开展了丰富多彩的课外活动，同学们的综合素质得到了提高。",
    "为了防止校园欺凌事件不再发生，学校加强了安全管理。"
  ],
  "answer": 2,
  "hints": [
    "💡 方向提示：本题考查病句辨析，注意检查主语、关联词、否定词",
    "📖 关键知识：「通过……使……」是典型的主语缺失；「能否」与「关键」不对应是两面对一面",
    "🎯 解题思路：A缺主语，B两面对一面，D否定失当（防止+不再=双重否定=肯定），只有C正确"
  ],
  "knowledgeCard": {

    "commonMistake": "A项「通过……使……」是中考高频考点，主语被介词掩盖；D项「防止……不再」双重否定变肯定",
    "relatedTopics": "语句衔接、语言得体"
  }
}
```

## 四、古诗文默写（fillblank）

### 格式要求
- 分为直接默写和理解性默写两种
- 直接默写：给上句填下句，或给下句填上句
- 理解性默写：给出主题/情感/手法的描述，填写对应诗句
- 答案中的易错字必须在知识点卡片中标注
- 多空题用中文分号「；」分隔答案

### 示例——直接默写

```json
{
  "id": 4,
  "type": "fillblank",
  "difficulty": 1,
  "question": "海日生残夜，______。（王湾《次北固山下》）",
  "answer": ["江春入旧年"],
  "hints": [

    "📖 关键知识：此联蕴含新事物孕育于旧事物之中的哲理",
    "🎯 解题思路：上句写「海日」对下句「江春」，「残夜」对「旧年」"
  ],
  "knowledgeCard": {
    "topic": "《次北固山下》名句默写",
```

## 十、提示设计原则

| 级别 | 内容要求 | 扣分 |
|------|---------|------|
| 提示1（方向） | 只告诉考哪个知识点方向或题目类型，不透露具体内容 | -30% |
| 提示2（知识） | 给出解题需要的核心知识点，但不直接指向答案 | -60% |
| 提示3（思路） | 给出排除法或推理过程，接近给出答案 | -90% |

## 十一、命题质量自检清单

每组题目生成后，按此清单自检：

- [ ] 知识点是否准确（对照教材）
- [ ] 题干表述是否清晰无歧义
- [ ] 选择题干扰项是否有合理迷惑性
- [ ] 正确答案分布是否均匀（A/B/C/D各约25%）
- [ ] 难度分布是否符合要求
- [ ] 每题是否都附有完整的知识点卡片和3级提示
- [ ] 知识点是否覆盖了不同考点（避免重复）
- [ ] 古诗文默写的答案中是否标注了易错字
- [ ] 材料分析题的设问是否由浅入深
- [ ] 文言文实词翻译是否准确
- [ ] 现代文阅读的评分标准是否清晰可操作
