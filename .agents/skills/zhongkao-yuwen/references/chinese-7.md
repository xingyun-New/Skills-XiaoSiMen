# 七年级语文（上册 + 下册）

> 人教版（部编版）七年级语文教材知识点框架。待补充完整教材原文时，请参照 [textbook-guide.md](textbook-guide.md) 的格式添加。

## 上册

### 第一单元：四季美景

#### 第1课 春（朱自清）
**核心知识点：**
1. 文体：写景抒情散文
2. 结构：盼春→绘春（春草图/春花图/春风图/春雨图/迎春图）→赞春
3. 修辞：比喻、拟人、排比

**必背名句：**
- "吹面不寒杨柳风"（引用志南和尚诗句）

**易错字词：**
- 酝酿(yùn niàng)、黄晕(yùn)、应和(hè)、嘹亮(liáo)、朗润(rùn)

#### 第2课 济南的冬天（老舍）
**核心知识点：**
1. 文体：写景散文
2. 特点：温晴——济南冬天的总特征
3. 写作顺序：山→水，由远到近

**易错字词：**
- 镶(xiāng)、贮蓄(zhù xù)、澄清(chéng)、髻(jì)

#### 第3课 雨的四季（刘湛秋）
**核心知识点：**
- 春雨美丽而娇媚、夏雨热烈而粗犷、秋雨端庄而沉思、冬雨自然而平静

#### 第4课 古代诗歌四首
**必背篇目：**
1. **《观沧海》**（曹操）—— 豪迈壮阔，统一天下的雄心
   - 名句：日月之行，若出其中；星汉灿烂，若出其里
2. **《闻王昌龄左迁龙标遥有此寄》**（李白）—— 友情关怀
   - 名句：我寄愁心与明月，随君直到夜郎西
3. **《次北固山下》**（王湾）—— 思乡之情
   - 名句：海日生残夜，江春入旧年
4. **《天净沙·秋思》**（马致远）—— 游子思乡
   - 名句：夕阳西下，断肠人在天涯

### 第二单元：至爱亲情

#### 第5课 秋天的怀念（史铁生）
- 母爱主题，双腿瘫痪后母亲的关爱
- 易错字：瘫痪(tān huàn)、憔悴(qiáo cuì)、诀别(jué)

#### 第6课 散步（莫怀戚）
- 家庭亲情，三代人的和谐
- 以小见大，"我"的责任担当

#### 第7课 散文诗两首
1. **《金色花》**（泰戈尔）—— 母子之爱
2. **《荷叶·母亲》**（冰心）—— 母爱

#### 第8课 《世说新语》二则
**必背篇目：**
1. **《咏雪》** —— "未若柳絮因风起"
2. **《陈太丘与友期行》** —— 诚信、方正

**重点文言词：**
- 期：约定 / 去：离开 / 委：丢下、舍弃 / 顾：回头看
- 尊君：对别人父亲的尊称 / 家君：对自己父亲的谦称

### 第三单元：学习生活

#### 第9课 从百草园到三味书屋（鲁迅）
- 出自《朝花夕拾》，对比百草园的快乐与三味书屋的枯燥
- 表达对童年的怀念和对束缚儿童天性的封建教育的不满

#### 第10课 再塑生命的人（海伦·凯勒）
- 记叙莎莉文老师教育"我"的过程

#### 第11课 《论语》十二章
**必背全文**

**重点句：**
- 学而时习之，不亦说乎
- 温故而知新，可以为师矣
- 学而不思则罔，思而不学则殆
- 三人行，必有我师焉
- 知之者不如好之者，好之者不如乐之者
- 逝者如斯夫，不舍昼夜
- 三军可夺帅也，匹夫不可夺志也
- 博学而笃志，切问而近思

**重点词：**
- 说(yuè)=悦 / 愠(yùn)：生气 / 罔(wǎng)：迷惑 / 殆(dài)：疑惑
- 好(hào)：喜爱 / 笃(dǔ)：坚定

## 名著阅读

### 《朝花夕拾》（鲁迅）
- **体裁**：回忆性散文集，原名《旧事重提》，共10篇
- **核心篇目**：
  - 《从百草园到三味书屋》：童年乐园与枯燥私塾的对比
  - 《阿长与〈山海经〉》：纪念长妈妈
  - 《藤野先生》：感恩日本老师的正直与热情
  - 《五猖会》：批判封建教育对儿童天性的压制
  - 《狗·猫·鼠》：讽刺"正人君子"
- **主题**：对童年和青年往事的深情回忆，对封建制度和思想的批判

### 《西游记》（吴承恩）
- **体裁**：古典长篇神魔小说
- **主要人物**：
  - 孙悟空：勇敢机智、嫉恶如仇、不畏强权
  - 唐僧：慈悲善良、意志坚定、不辨是非
  - 猪八戒：好吃懒做、但忠心重情
  - 沙僧：任劳任怨、忠厚老实
- **经典情节**：大闹天宫、三打白骨精、真假美猴王、火焰山借扇
- **主题**：坚持不懈、团队协作、正义终将战胜邪恶

### 《骆驼祥子》（老舍）
- **体裁**：现实主义长篇小说
- **主人公**：祥子，北平人力车夫
- **核心情节**：祥子三起三落
  - 一起：苦干三年买了新车 → 一落：被大兵抢走
  - 二起：卖骆驼攒钱 → 二落：被孙侦探敲诈
  - 三起：靠虎妞买了车 → 三落：虎妞难产卖车安葬
- **结局**：祥子从勤劳上进堕落为麻木潦倒的行尸走肉
- **主题**：旧社会对底层劳动者的压迫与毁灭

### 《海底两万里》（儒勒·凡尔纳）
- **体裁**：科幻小说
- **主要人物**：尼摩船长——神秘、博学、正义感、仇恨殖民压迫
- **载体**：鹦鹉螺号潜水艇
- **经典情节**：海底森林狩猎、搁浅土人围攻、采珠人遇鲨鱼、南极冰山困境
- **主题**：科学探索精神，对海洋的热爱，反殖民压迫
