# 教材内容添加指南

本 skill 的出题精度取决于教材知识点的完整程度。请按以下方式补充各科教材内容。

## 文件对应关系

| 文件 | 对应教材 |
|------|---------|

## 推荐内容格式

每个教材文件建议按以下结构组织：

```markdown
# X年级语文（上册 + 下册）

## 上册

### 第一单元：[单元主题]

#### 第1课 [课文名称]（作者）

**核心知识点：**
1. 文体：[散文/小说/议论文/说明文/诗歌]
2. 主题：[核心主题]
3. 写作手法：[重要的表达技巧]

**必背名句：**
- [需要背诵的重要句子]

**易错字词：**
- [字词(注音)]

**重点概念：**
- [需要掌握的概念]

---

### 文言文/古诗词

#### [篇名]（作者）
**必背全文**

**名句：**
- [重要名句]

**重点词：**
- [文言实词/虚词解释]

**特殊句式：**
- [倒装/省略/判断/被动]

---

### 名著阅读

#### 《[书名]》（作者）
- 体裁：[类型]
- 主要人物：[人物+性格特点]
- 经典情节：[重要情节]
- 主题：[核心主题]
```

## 重点内容优先级

### 必须优先补充的内容

1. **古诗文原文**：61篇必背古诗文的完整原文和注释
2. **文言文实词**：每篇文言文的重点实词、虚词释义
3. **名著核心情节**：12部必读名著的人物、情节、主题
4. **易错字词表**：中考高频易错字音、字形汇总

### 建议补充的内容

5. 现代文阅读经典篇目的主题分析
6. 各年级作文素材和范文片段
7. 中考真题考点分布统计

## 内容来源建议

1. **电子课本原文**：直接复制教材正文中的核心段落
2. **课后习题**：课本上的"思考探究"和"积累拓展"
3. **教师用书要点**：教学重点、难点提示
4. **中考考纲**：当地中考考试大纲中的具体考点清单

## 注意事项

- 每个文件建议控制在合理长度内，过长的内容可拆分
- 优先补充高频考点和易错知识点
- 如果教材版本不是人教版（部编版），请在文件开头注明版本
- 古诗文原文务必核对准确，一字之差影响默写得分
- 内容更新后，skill 会自动使用最新知识点出题
