# 游戏化机制

交互练习页面的积分、成就和激励系统。

## 积分规则

| 事件 | 基础分 | 说明 |
|------|--------|------|
| 答对（基础题） | 10 + 5 | 基础分 + 难度加成 |
| 答对（中等题） | 10 + 10 | 基础分 + 难度加成 |
| 答对（拔高题） | 10 + 15 | 基础分 + 难度加成 |
| 连击3次 | +2/题 | 基础分提升至12 |
| 连击5次 | +5/题 | 基础分提升至15 |
| 答错 | 0 | 连击归零 |

### 提示扣分机制

使用提示后，答对得分 = 基础分 × 系数：

| 使用提示数 | 得分系数 | 示例（中等题满分20分） |
|-----------|---------|---------------------|
| 0个提示 | ×1.0（满分） | 20分 |
| 1个提示 | ×0.7（-30%） | 14分 |
| 2个提示 | ×0.4（-60%） | 8分 |
| 3个提示 | ×0.1（-90%） | 2分 |

设计理念：
- 鼓励先独立思考再看提示
- 即使用了3个提示答对也有分数（奖励坚持做题）
- 使用提示不影响连击计数（但成就动画只在0提示时触发）

## 结果评级

| 正确率 | 评级 | 建议 |
|--------|------|------|
| ≥ 90% | 🏆 优秀 | 挑战更高难度 |
| 70%-89% | 👍 良好 | 巩固薄弱知识点 |
| 60%-69% | ✓ 及格 | 重点复习错题 |
| < 60% | 💪 加油 | 回顾课本基础内容 |

## 错题重练

做完全部题目后，如果有错题：
1. 汇总所有错题涉及的知识点（去重），显示为"薄弱知识点"
2. 提供"错题重练"按钮，只重新练习答错的题目
3. 重练时积分和连击重新计算

## 视觉反馈

- **答对**：选项变绿 + 知识点卡片（绿色背景）+ 公式展示
- **答错**：选项变红，正确选项变绿 + 知识点卡片（橙色背景，含易错提醒和公式）
- **连击成就**：右上角弹出金色成就徽章，3秒后自动消失
- **进度条**：顶部绿色进度条实时更新

## 计算题/实验题自评

计算题和实验探究题采用自评机制：
1. 学生作答后，系统显示参考答案和评分标准
2. 学生自评：「完全正确」/「部分正确」/「错误」
3. 完全正确得满分，部分正确得50%，错误得0分
4. 自评结果计入总成绩和连击系统
