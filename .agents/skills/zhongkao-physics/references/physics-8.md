# 八年级物理（上册 + 下册）

人教版初中物理八年级教材知识点框架。

## 上册

### 第一章：机械运动

#### 第1节 长度和时间的测量
**核心知识点：**
1. 长度单位：国际单位 m（米），常用单位 km、dm、cm、mm、μm、nm
2. 换算关系：1km = 10³m，1m = 10dm = 100cm = 1000mm，1mm = 10³μm，1μm = 10³nm
3. 刻度尺使用：认清量程和分度值，紧贴被测物体，视线与刻度垂直，读数要估读
4. 时间单位：s（秒），1h = 60min = 3600s
5. 误差：测量值与真实值之间的差异，不可消除，只能减小（多次测量取平均值）

**易错点：**
- 误差≠错误，误差不可避免，错误可以避免
- 刻度尺读数必须估读到分度值的下一位

#### 第2节 运动的描述
**核心知识点：**
1. 机械运动：物体位置随时间的变化
2. 参照物：判断物体是否运动时选取的标准
3. 运动和静止的相对性：同一物体相对不同参照物，运动状态可能不同

#### 第3节 运动的快慢
**核心公式：**
- 速度公式：v = s/t （v：速度 m/s，s：路程 m，t：时间 s）
- 变形公式：s = vt，t = s/v
- 单位换算：1m/s = 3.6km/h

**核心知识点：**
1. 匀速直线运动：沿直线运动，速度大小不变
2. 变速运动：速度大小变化，用平均速度描述 v̄ = s总/t总

**易错点：**
- km/h 与 m/s 的换算：×3.6 或 ÷3.6
- 平均速度 ≠ 速度的平均值

### 第二章：声现象

#### 第1节 声音的产生与传播
**核心知识点：**
1. 声音的产生：物体振动产生声音
2. 声音的传播：需要介质（固体、液体、气体），真空不能传声
3. 声速：空气中约 340m/s（15℃），固体中最快，液体次之，气体最慢
4. 回声：声音遇到障碍物被反射回来（距障碍物 ≥17m 才能区分回声和原声）

#### 第2节 声音的特性
**核心知识点：**
1. 音调：声音的高低，由振动频率决定（频率高→音调高）
2. 响度：声音的大小（强弱），由振幅决定（振幅大→响度大）
3. 音色：声音的品质，由发声体的材料和结构决定

**易错点：**
- 音调和响度容易混淆："高"可能指音调高也可能指响度大
- 频率单位：赫兹（Hz），人耳听觉范围 20Hz~20000Hz

#### 第3节 声的利用
**核心知识点：**
1. 声传递信息：B超、声呐、回声定位
2. 声传递能量：超声波清洗、碎石

#### 第4节 噪声的危害和控制
**核心知识点：**
1. 噪声控制三途径：在声源处减弱、在传播过程中减弱、在人耳处减弱

### 第三章：物态变化

#### 第1节 温度
**核心知识点：**
1. 温度：表示物体冷热程度
2. 摄氏温标：冰水混合物 0℃，标准大气压下沸水 100℃
3. 温度计原理：液体的热胀冷缩
4. 体温计：35℃~42℃，分度值 0.1℃，可以离开人体读数

#### 第2节 熔化和凝固
**核心知识点：**
1. 熔化（固→液，吸热）、凝固（液→固，放热）
2. 晶体有固定的熔点/凝固点（冰、海波等），非晶体没有（蜡、玻璃等）
3. 晶体熔化条件：达到熔点 + 持续吸热

#### 第3节 汽化和液化
**核心知识点：**
1. 汽化（液→气，吸热）：蒸发和沸腾两种方式
2. 蒸发：液体表面，任何温度都能发生，有致冷作用
3. 沸腾：液体内部和表面同时发生，达到沸点且持续吸热
4. 液化（气→液，放热）：降低温度或压缩体积

#### 第4节 升华和凝华
**核心知识点：**
1. 升华（固→气，吸热）：如樟脑丸变小、冰冻衣服变干
2. 凝华（气→固，放热）：如霜、雾凇、冰花

**记忆口诀：** 「固液气，吸热升，放热降」——从左到右吸热，从右到左放热

### 第四章：光现象

#### 第1节 光的直线传播
**核心知识点：**
1. 光在同种均匀介质中沿直线传播
2. 光速：真空中 c = 3×10⁸ m/s
3. 应用：影子、日食月食、小孔成像

#### 第2节 光的反射
**核心知识点：**
1. 反射定律：反射光线与入射光线、法线在同一平面内；反射光线和入射光线分居法线两侧；反射角等于入射角
2. 镜面反射与漫反射：都遵守反射定律
3. 光路可逆

#### 第3节 平面镜成像
**核心知识点：**
1. 特点：等大、等距、虚像、连线与镜面垂直
2. 成像原理：光的反射

#### 第4节 光的折射
**核心知识点：**
1. 折射规律：折射光线、入射光线、法线在同一平面内
2. 光从空气斜射入水/玻璃：折射角 < 入射角
3. 光从水/玻璃斜射入空气：折射角 > 入射角
4. 垂直入射时不发生偏折

#### 第5节 光的色散
**核心知识点：**
1. 白光由红橙黄绿蓝靛紫七色光组成
2. 三原色（光）：红、绿、蓝
3. 不透明物体的颜色由它反射的色光决定
4. 透明物体的颜色由它透过的色光决定

---

### 第五章：透镜及其应用

#### 第1节 透镜
**核心知识点：**
1. 凸透镜：中间厚、边缘薄，对光有会聚作用
2. 凹透镜：中间薄、边缘厚，对光有发散作用
3. 焦点与焦距：平行于主光轴的光通过凸透镜后会聚于焦点F，焦点到光心的距离为焦距f

#### 第2节 生活中的透镜
**核心知识点：**
1. 照相机：物距 > 2f，成倒立缩小实像
2. 投影仪：f < 物距 < 2f，成倒立放大实像
3. 放大镜：物距 < f，成正立放大虚像

#### 第3节 凸透镜成像的规律
**核心知识点（凸透镜成像规律表）：**

| 物距 u | 像的性质 | 像距 v | 应用 |
|--------|---------|--------|------|
| u > 2f | 倒立缩小实像 | f < v < 2f | 照相机 |
| u = 2f | 倒立等大实像 | v = 2f | 测焦距 |
| f < u < 2f | 倒立放大实像 | v > 2f | 投影仪 |
| u = f | 不成像 | — | 平行光源 |
| u < f | 正立放大虚像 | — | 放大镜 |

**记忆口诀：** 「一焦分虚实，二焦分大小；实倒虚正，物近像远像变大」

#### 第4节 眼睛和眼镜
**核心知识点：**
1. 近视：晶状体太厚/眼球前后径过长 → 像成在视网膜前 → 戴凹透镜矫正
2. 远视：晶状体太薄/眼球前后径过短 → 像成在视网膜后 → 戴凸透镜矫正

### 第六章：质量与密度

#### 第1节 质量
**核心知识点：**
1. 质量：物体所含物质的多少，符号 m
2. 单位：kg（千克），1t = 10³kg，1kg = 10³g，1g = 10³mg
3. 质量是物体的属性，不随位置、形状、状态的改变而改变
4. 天平的使用：水平放置、调平衡、左物右码

#### 第2节 密度
**核心公式：**
- 密度公式：ρ = m/V （ρ：密度 kg/m³，m：质量 kg，V：体积 m³）
- 变形公式：m = ρV，V = m/ρ
- 单位换算：1g/cm³ = 10³ kg/m³

**核心知识点：**
1. 密度是物质的属性，同种物质密度相同（温度、状态不变时）
2. 水的密度：ρ水 = 1.0×10³ kg/m³ = 1g/cm³

#### 第3节 测量物质的密度
**核心知识点：**
1. 原理：ρ = m/V
2. 固体密度测量：天平测质量 + 量筒测体积（排水法）
3. 液体密度测量：天平测空杯质量→倒入液体测总质量→量筒测体积

**易错点：**
- 测量顺序很重要：先测质量再测体积（避免沾水导致质量不准）
- 不规则固体用排水法测体积

## 下册

### 第七章：力

#### 第1节 力
**核心知识点：**
1. 力的定义：物体对物体的作用
2. 力的作用是相互的
3. 力的作用效果：改变物体的形状、改变物体的运动状态
4. 力的三要素：大小、方向、作用点
5. 力的单位：牛顿（N），托起两个鸡蛋约需 1N

#### 第2节 弹力
**核心知识点：**
1. 弹力：物体由于发生弹性形变而产生的力
2. 弹簧测力计原理：在弹性限度内，弹簧伸长量与所受拉力成正比

#### 第3节 重力
**核心公式：**
- G = mg （G：重力 N，m：质量 kg，g ≈ 9.8N/kg ≈ 10N/kg）

**核心知识点：**
1. 重力方向：竖直向下
2. 重心：重力的作用点（质量均匀分布的物体在几何中心）

---

### 第八章：运动和力

#### 第1节 牛顿第一定律
**核心知识点：**
1. 牛顿第一定律（惯性定律）：一切物体在没有受到力的作用时，总保持静止状态或匀速直线运动状态
2. 惯性：物体保持原来运动状态不变的性质，惯性大小只与质量有关

#### 第2节 二力平衡
**核心知识点：**
1. 平衡力条件：等大、反向、共线、同体（大小相等、方向相反、在同一直线上、作用在同一物体上）
2. 平衡状态：静止或匀速直线运动

**易错点：**
- 平衡力 vs 相互作用力：平衡力在同一物体上，相互作用力在不同物体上

#### 第3节 摩擦力
**核心知识点：**
1. 滑动摩擦力影响因素：压力大小和接触面粗糙程度
2. 增大摩擦：增大压力、增大接触面粗糙程度
3. 减小摩擦：减小压力、使接触面光滑、变滑动为滚动、加润滑油

---

### 第九章：压强

#### 第1节 压强
**核心公式：**
- p = F/S （p：压强 Pa，F：压力 N，S：受力面积 m²）
- 1Pa = 1N/m²

**核心知识点：**
1. 压力：垂直压在物体表面上的力（不一定等于重力！）
2. 增大压强：增大压力或减小受力面积
3. 减小压强：减小压力或增大受力面积

#### 第2节 液体的压强
**核心公式：**
- p = ρgh （p：液体压强 Pa，ρ：液体密度 kg/m³，g ≈ 10N/kg，h：深度 m）

**核心知识点：**
1. 液体压强与液体密度和深度有关，与容器形状无关
2. 同一深度，液体向各个方向的压强相等
3. 连通器：液体不流动时，各容器中液面相等

#### 第3节 大气压强
**核心知识点：**
1. 大气压强的存在：马德堡半球实验
2. 标准大气压：p₀ = 1.013×10⁵ Pa = 760mmHg
3. 大气压随高度增加而减小

#### 第4节 流体压强与流速的关系
**核心知识点：**
1. 伯努利原理：流速大的地方压强小，流速小的地方压强大
2. 应用：飞机升力、喷雾器

### 第十章：浮力

#### 第1节 浮力
**核心公式：**
- F浮 = ρ液gV排 （阿基米德原理）
- F浮 = G - F示 （称重法）
- 漂浮/悬浮时：F浮 = G物

**核心知识点：**
1. 浮力方向：竖直向上
2. 浮力产生原因：上下表面的压力差
3. 阿基米德原理：浸在液体中的物体受到向上的浮力，大小等于排开液体所受的重力

#### 第2节 物体的浮沉条件
**核心知识点：**
| 条件 | 运动状态 | 密度关系 |
|------|---------|---------|
| F浮 > G | 上浮 | ρ物 < ρ液 |
| F浮 = G | 悬浮 | ρ物 = ρ液 |
| F浮 < G | 下沉 | ρ物 > ρ液 |
| 漂浮 | 静止在液面 | ρ物 < ρ液 |

**易错点：**
- 漂浮时 V排 < V物，F浮 = G物
- 悬浮和漂浮都有 F浮 = G，但悬浮是 V排 = V物

### 第十一章：功和机械能

#### 第1节 功
**核心公式：**
- W = Fs （W：功 J，F：力 N，s：在力的方向上移动的距离 m）
- 1J = 1N·m

**核心知识点：**
1. 做功的两个必要因素：有力作用在物体上 + 物体在力的方向上移动了距离
2. 不做功的三种情况：有力无距离、有距离无力、力与运动方向垂直

#### 第2节 功率
**核心公式：**
- P = W/t （P：功率 W，W：功 J，t：时间 s）
- P = Fv （推导公式）
- 1kW = 1000W

#### 第3节 动能和势能
**核心知识点：**
1. 动能：物体由于运动而具有的能量，与质量和速度有关
2. 重力势能：与质量和高度有关
3. 弹性势能：与弹性形变程度有关
4. 机械能 = 动能 + 势能

#### 第4节 机械能及其转化
**核心知识点：**
1. 动能和势能可以互相转化
2. 在只有重力做功的情况下，机械能守恒

---

### 第十二章：简单机械

#### 第1节 杠杆
**核心公式：**
- 杠杆平衡条件：F₁ × L₁ = F₂ × L₂

**核心知识点：**
1. 五要素：支点、动力、阻力、动力臂、阻力臂
2. 省力杠杆：动力臂 > 阻力臂（费距离）
3. 费力杠杆：动力臂 < 阻力臂（省距离）
4. 等臂杠杆：动力臂 = 阻力臂（天平）

#### 第2节 滑轮
**核心知识点：**
1. 定滑轮：改变力的方向，不省力
2. 动滑轮：省一半力，但不改变方向
3. 滑轮组：n段绳子承担物重，F = (G物+G动)/n

#### 第3节 机械效率
**核心公式：**
- η = W有用/W总 × 100%
- η 总是 < 100%

**核心知识点：**
1. 有用功：对我们有用的功
2. 额外功：克服摩擦力和提升机械自身重力所做的功
3. 总功 = 有用功 + 额外功
