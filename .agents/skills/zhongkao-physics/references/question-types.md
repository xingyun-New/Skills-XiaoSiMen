# 物理题型模板与示例

所有题型的标准格式、命题要求和完整示例。

## 一、选择题

### 格式要求
- 4个选项（A/B/C/D），有且仅有1个正确答案
- 题干表述清晰无歧义，物理量必须带单位
- 四个选项长度接近，语法结构一致
- 正确答案在A/B/C/D中均匀分布
- 涉及计算时，干扰项应来源于常见计算错误（如未换算单位、公式用错等）

### 模板

```markdown
### 第X题（选择题 | 难度：★★☆）

**【题目】** [题干内容，包含必要的物理情境和数据]

A. [选项A]
B. [选项B]
C. [选项C]
D. [选项D]

**【提示】**
- 💡 提示1（方向）：[指出本题考查的知识范围]
- 📖 提示2（知识）：[给出关键公式或定律]
- 🎯 提示3（思路）：[给出解题思路，接近答案]

**【答案】** [正确选项字母]

**【知识点卡片】**
- 📚 考点：[具体知识点名称]
- 🔬 相关公式：[解题涉及的公式]
- 📖 出处：人教版X年级X册 第X章
- 💡 关键记忆：[一句话核心记忆]
- ⚠️ 易错提醒：[说明干扰项的迷惑原因和常见错误]
- 🔗 关联知识：[相关的其他知识点]
```

### 提示设计原则

| 级别 | 内容要求 | 扣分 |
|------|---------|------|
| 提示1（方向） | 只告诉考哪个章节/知识点方向，不透露具体公式 | -30% |
| 提示2（知识） | 给出解题需要的核心公式或定律，但不直接指向答案 | -60% |
| 提示3（思路） | 给出排除法或计算过程提示，接近给出答案 | -90% |

### 示例（力学）

```markdown
### 第1题（选择题 | 难度：★★☆）

**【题目】** 一个质量为 500g 的物体，放在水平桌面上静止不动。下列说法正确的是：

A. 物体受到的重力和桌面对它的支持力是一对相互作用力
B. 物体受到的重力大小约为 5N
C. 物体对桌面的压力就是物体的重力
D. 物体没有受到摩擦力说明桌面是光滑的

**【答案】** B

**【知识点卡片】**
- 📚 考点：重力的计算、平衡力与相互作用力的区别
- 🔬 相关公式：G = mg，g ≈ 10N/kg
- 📖 出处：人教版八年级下册 第七章 力
- 💡 关键记忆：G = mg = 0.5kg × 10N/kg = 5N；平衡力作用在同一物体上，相互作用力作用在两个物体上
- ⚠️ 易错提醒：A选项中重力和支持力是「平衡力」而非相互作用力，它们作用在同一物体上；C选项中压力和重力大小相等但不是同一个力
- 🔗 关联知识：牛顿第三定律、二力平衡条件
```

## 二、判断题

### 格式要求
- 陈述句形式，表述完整
- 错误题目要有明确的可辨别错误点
- 正确和错误题目各占约50%

### 模板

```markdown
### 第X题（判断题 | 难度：★☆☆）

**【题目】** [陈述句]（  ）

**【答案】** ✓正确 / ✗错误

**【知识点卡片】**
- 📚 考点：[知识点]
- 📖 出处：[教材位置]
- 💡 关键记忆：[正确表述]
- ⚠️ 易错提醒：[如果是错误题，说明正确的应该是什么]
```

### 示例（电学）

```markdown
### 第3题（判断题 | 难度：★☆☆）

**【题目】** 电路中电流的方向是从电源的负极流向正极。（  ）

**【答案】** ✗错误

**【知识点卡片】**
- 📚 考点：电流方向的规定
- 📖 出处：人教版九年级全一册 第十五章 电流和电路
- 💡 关键记忆：规定正电荷定向移动的方向为电流方向，在外电路中从正极流向负极
- ⚠️ 易错提醒：电子移动方向与电流方向相反！电子在外电路中从负极流向正极，但电流方向相反
```

## 三、填空题

### 格式要求
- 每空只有唯一标准答案（或已列明的等价答案）
- 空格用下划线表示，标注每空分值
- 物理量填空必须包含单位（除非题目已给出单位）
- 避免一个句子中挖空过多导致无法理解句意

### 模板

```markdown
### 第X题（填空题 | 难度：★☆☆）

**【题目】** [含有______的陈述句]。

**【答案】** ① ______ ② ______

**【知识点卡片】**
- 📚 考点：[知识点]
- 🔬 相关公式：[涉及的公式]
- 📖 出处：[教材位置]
- 💡 关键记忆：[包含答案的完整句子]
```

### 示例（光学）

```markdown
### 第5题（填空题 | 难度：★☆☆）

**【题目】** 光在真空中的传播速度约为______m/s；光从空气斜射入水中时，折射角______入射角（填"大于""等于"或"小于"）。

**【答案】** ① 3×10⁸ ② 小于

**【知识点卡片】**
- 📚 考点：光速、光的折射规律
- 📖 出处：人教版八年级上册 第四章 光现象、第五章 透镜及其应用
- 💡 关键记忆：光速 c = 3×10⁸ m/s；光从光疏介质进入光密介质时折射角小于入射角
- 🔗 关联知识：折射定律、全反射（高中拓展）
```

## 四、计算题

### 格式要求
- 提供完整的已知条件，数据合理
- 明确求解目标
- 参考答案必须写出完整的「已知—求—解—答」过程
- 每步计算标注使用的公式
- 注意单位换算和有效数字

### 示例（电学计算）

```markdown
### 第9题（计算题 | 难度：★★★ | 6分）

**【题目】** 如图所示电路，电源电压为 6V，R₁ = 10Ω，R₂ = 20Ω，R₁ 和 R₂ 串联。求：
（1）电路中的电流；（3分）
（2）R₂ 两端的电压。（3分）

**【参考解答】**
已知：U = 6V，R₁ = 10Ω，R₂ = 20Ω，R₁ 和 R₂ 串联
求：（1）I；（2）U₂

解：
（1）串联电路总电阻 R = R₁ + R₂ = 10Ω + 20Ω = 30Ω
    由欧姆定律 I = U/R = 6V ÷ 30Ω = 0.2A

（2）由欧姆定律 U₂ = IR₂ = 0.2A × 20Ω = 4V

答：（1）电路中的电流为 0.2A；（2）R₂ 两端的电压为 4V。

**【知识点卡片】**
- 📚 考点：串联电路特点、欧姆定律计算
- 🔬 核心公式：R串 = R₁+R₂，I = U/R，U = IR
- 📖 出处：人教版九年级全一册 第十七章 欧姆定律
- 💡 解题技巧：串联电路三步走——①求总电阻 ②用总电压和总电阻求电流 ③用电流和分电阻求分电压
- ⚠️ 易错提醒：串联电路电流处处相等，不要对各电阻分别求电流；电压有分压关系 U = U₁ + U₂
```

## 六、命题质量自检清单

每组题目生成后，按此清单自检：

- [ ] 物理概念和定律是否准确
- [ ] 物理量是否都带有正确的单位
- [ ] 数据是否合理（符合实际物理范围）
- [ ] 计算题的参考答案是否正确、过程完整
- [ ] 题干表述是否清晰无歧义
- [ ] 选择题干扰项是否来源于常见计算/概念错误
- [ ] 正确答案分布是否均匀（A/B/C/D各约25%）
- [ ] 难度分布是否符合要求
- [ ] 每题是否都附有完整的知识点卡片（含公式）
- [ ] 知识点是否覆盖了不同考点（避免重复）
- [ ] 实验题是否体现了控制变量法等科学方法
- [ ] 单位换算是否正确（如 cm→m, g→kg, cm³→m³）
- [ ] 易错提醒是否切中学生真实痛点
