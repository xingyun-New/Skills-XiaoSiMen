# 九年级物理（全一册）

人教版初中物理九年级教材知识点框架。

## 第十三章：内能

### 第1节 分子热运动
**核心知识点：**
1. 物质由分子组成，分子在不停地做无规则运动
2. 扩散现象：不同物质相互接触时彼此进入对方（温度越高扩散越快）
3. 分子间存在引力和斥力

### 第2节 内能
**核心知识点：**
1. 内能：物体内所有分子热运动的动能与分子势能的总和
2. 内能与温度：温度升高，内能增大；温度降低，内能减小
3. 改变内能的两种方式：做功、热传递（等效的）

**易错点：**
- 0℃的物体也有内能（分子仍在运动）
- 内能大小与温度、质量、状态都有关

### 第3节 比热容
**核心公式：**
- Q吸 = cm(t - t₀) = cmΔt
- Q放 = cm(t₀ - t) = cmΔt
- c：比热容 J/(kg·℃)

**核心知识点：**
1. 比热容：单位质量的某种物质温度升高1℃吸收的热量
2. 水的比热容最大：c水 = 4.2×10³ J/(kg·℃)
3. 比热容是物质的属性，不随质量、温度变化而变化

**易错点：**
- 注意 t₀ 是初温，t 是末温，吸热时 t > t₀，放热时 t₀ > t
- 热量只能「吸收」或「放出」，不能说「含有」或「具有」

## 第十四章：内能的利用

### 第1节 热机
**核心知识点：**
1. 内燃机工作过程：吸气→压缩→做功→排气（四冲程）
2. 压缩冲程：机械能→内能
3. 做功冲程：内能→机械能（唯一对外做功的冲程）
4. 一个工作循环：4个冲程，曲轴转2圈，对外做功1次

### 第2节 热机的效率
**核心公式：**
- 热值：q = Q放/m → Q放 = mq（完全燃烧）
- 热机效率：η = W有用/Q放 × 100%

**核心知识点：**
1. 热值：1kg 某种燃料完全燃烧放出的热量
2. 热值是燃料的属性，不随质量变化
3. 热机效率 < 100%（有废气带走的热量、克服摩擦等）

### 第3节 能量的转化和守恒
**核心知识点：**
1. 能量守恒定律：能量既不会凭空消灭，也不会凭空产生，只会从一种形式转化为另一种形式
2. 永动机不可能实现

## 第十五章：电流和电路

### 第1节 两种电荷
**核心知识点：**
1. 正电荷：用丝绸摩擦过的玻璃棒带正电
2. 负电荷：用毛皮摩擦过的橡胶棒带负电
3. 同种电荷互相排斥，异种电荷互相吸引
4. 验电器：检验物体是否带电（金属箔张开）

### 第2节 电流和电路
**核心知识点：**
1. 电流方向：规定正电荷定向移动的方向为电流方向（外电路从正极到负极）
2. 电路组成：电源、导线、开关、用电器
3. 通路、断路、短路

### 第3节 串联和并联
**核心知识点：**
1. 串联：用电器依次连接，电流只有一条路径
2. 并联：用电器并列连接，电流有多条路径
3. 串联电路特点：电流处处相等
4. 并联电路特点：各支路两端电压相等

### 第4节 电流的测量
**核心知识点：**
1. 电流单位：安培（A），1A = 1000mA
2. 电流表使用：串联在电路中，正接线柱接正极，选择合适量程
3. 量程：0~0.6A（分度值0.02A）、0~3A（分度值0.1A）

### 第5节 串、并联电路中电流的规律
**核心知识点：**
1. 串联电路：I = I₁ = I₂（电流处处相等）
2. 并联电路：I = I₁ + I₂（干路电流等于各支路电流之和）

## 第十六章：电压 电阻

### 第1节 电压
**核心知识点：**
1. 电压使电路中形成电流，电源提供电压
2. 电压单位：伏特（V），1kV = 10³V，1V = 10³mV
3. 常见电压值：一节干电池 1.5V、蓄电池 2V、家庭电路 220V、安全电压 ≤36V

### 第2节 串、并联电路中电压的规律
**核心知识点：**
1. 串联电路：U = U₁ + U₂（总电压等于各用电器两端电压之和）
2. 并联电路：U = U₁ = U₂（各支路两端电压相等）

### 第3节 电阻
**核心知识点：**
1. 电阻：导体对电流的阻碍作用
2. 电阻单位：欧姆（Ω），1kΩ = 10³Ω，1MΩ = 10⁶Ω
3. 影响电阻的因素：材料、长度、横截面积、温度
4. 导体电阻是导体本身的属性，与电压和电流无关

### 第4节 变阻器
**核心知识点：**
1. 滑动变阻器原理：通过改变接入电路的电阻丝长度来改变电阻
2. 连接方法：「一上一下」接两个接线柱
3. 使用前应将滑片调到阻值最大处

---

## 第十七章：欧姆定律

### 第1节 电流与电压和电阻的关系
**核心知识点：**
1. 电流与电压成正比（电阻不变时）
2. 电流与电阻成反比（电压不变时）
3. 实验中用滑动变阻器控制电压或电流

### 第2节 欧姆定律
**核心公式：**
- I = U/R （I：电流 A，U：电压 V，R：电阻 Ω）
- 变形公式：U = IR，R = U/I

**核心知识点：**
1. 欧姆定律：导体中的电流与导体两端的电压成正比，与导体的电阻成反比
2. R = U/I 是电阻的测量公式，不是定义公式，电阻不随U、I变化

### 第3节 电阻的测量
**核心知识点：**
1. 伏安法测电阻：原理 R = U/I
2. 器材：电压表、电流表、滑动变阻器、电源、开关、导线、待测电阻
3. 滑动变阻器作用：保护电路 + 改变待测电阻两端电压（多次测量取平均值）

### 第4节 欧姆定律在串、并联电路中的应用
**核心公式：**
- 串联：R总 = R₁ + R₂
- 并联：1/R总 = 1/R₁ + 1/R₂ → R总 = R₁R₂/(R₁+R₂)

**核心知识点：**
1. 串联分压：U₁/U₂ = R₁/R₂
2. 并联分流：I₁/I₂ = R₂/R₁

**易错点：**
- 串联电路中电阻大的分到的电压大
- 并联电路中电阻小的通过的电流大

## 第十八章：电功率

### 第1节 电能 电功
**核心公式：**
- W = UIt = Pt
- 1kW·h = 3.6×10⁶ J
- 电能表读数：最后一位是小数

### 第2节 电功率
**核心公式：**
- P = W/t = UI
- 推导公式：P = I²R = U²/R
- 1kW = 1000W

**核心知识点：**
1. 额定功率：用电器在额定电压下的功率
2. 实际功率：用电器在实际电压下的功率
3. 实际电压 > 额定电压时，实际功率 > 额定功率（可能烧毁）
4. 实际电压 < 额定电压时，实际功率 < 额定功率（不能正常工作）

### 第3节 测量小灯泡的电功率
**核心知识点：**
1. 原理：P = UI
2. 器材：电压表、电流表、滑动变阻器、电源、小灯泡、开关、导线
3. 滑动变阻器作用：保护电路 + 调节小灯泡两端电压

### 第4节 焦耳定律
**核心公式：**
- Q = I²Rt

**核心知识点：**
1. 焦耳定律：电流通过导体产生的热量与电流的平方成正比、与电阻成正比、与通电时间成正比
2. 纯电阻电路：Q = W = UIt = I²Rt = U²t/R
3. 电热器：利用电流的热效应

## 第十九章：生活用电

### 第1节 家庭电路
**核心知识点：**
1. 家庭电路电压：220V
2. 进户线：火线和零线
3. 火线与零线之间电压 220V，零线与大地之间电压 0V
4. 保险丝/空气开关：电流过大时自动切断电路

### 第2节 家庭电路中电流过大的原因
**核心知识点：**
1. 短路：电流不经过用电器直接从火线流向零线
2. 用电器总功率过大：I = P总/U

### 第3节 安全用电
**核心知识点：**
1. 安全电压：不高于 36V
2. 触电类型：单线触电、双线触电
3. 三孔插座：多一根接地线，保护用电安全

## 第二十章：电与磁

### 第1节 磁现象 磁场
**核心知识点：**
1. 磁极：N极（北极）、S极（南极）
2. 磁极间的相互作用：同名磁极互相排斥，异名磁极互相吸引
3. 磁场：磁体周围存在的场，看不见摸不着，但客观存在
4. 磁感线：从N极出发回到S极（外部）

### 第2节 电生磁
**核心知识点：**
1. 奥斯特实验：通电导线周围存在磁场（电流的磁效应）
2. 电磁铁：通电产生磁性，断电磁性消失
3. 电磁铁的磁性强弱与电流大小和线圈匝数有关

### 第3节 电磁铁 电磁继电器
**核心知识点：**
1. 电磁继电器：利用电磁铁实现用低压控制高压（或弱电控制强电）
2. 应用：自动控制电路

### 第4节 电动机
**核心知识点：**
1. 原理：通电导体在磁场中受力
2. 能量转化：电能→机械能
3. 换向器的作用：使线圈持续转动

### 第5节 磁生电
**核心知识点：**
1. 电磁感应：闭合电路的一部分导体在磁场中做切割磁感线运动时，导体中就会产生电流
2. 感应电流方向与导体运动方向和磁场方向有关
3. 发电机原理：电磁感应，机械能→电能

## 第二十一章：信息的传递

### 第1节 现代顺风耳——电话
**核心知识点：**
1. 电话原理：话筒把声音转化为变化的电流，听筒把变化的电流转化为声音
2. 模拟信号和数字信号

### 第2节 电磁波的海洋
**核心知识点：**
1. 电磁波：变化的电流产生电磁波
2. 电磁波可以在真空中传播
3. 电磁波速度 = 光速：c = 3×10⁸ m/s
4. 波速、波长、频率关系：c = λf

### 第3节 广播、电视和移动通信
**核心知识点：**
1. 广播和电视通过电磁波传递信息
2. 移动通信依靠基站中转

### 第4节 越来越宽的信息之路
**核心知识点：**
1. 光纤通信：利用光的全反射传输信号
2. 卫星通信：地球同步轨道卫星

## 第二十二章：能源与可持续发展

### 第1节 能源
**核心知识点：**
1. 一次能源：直接从自然界获取（煤、石油、天然气、风能、太阳能、核能等）
2. 二次能源：一次能源加工转换得到（电能等）
3. 可再生能源 vs 不可再生能源

### 第2节 核能
**核心知识点：**
1. 裂变：重核分裂为较轻的核，释放大量能量（原子弹、核电站）
2. 聚变：轻核结合成较重的核，释放更多能量（氢弹、太阳）

### 第3节 太阳能
**核心知识点：**
1. 太阳能的利用：光热转换（太阳能热水器）、光电转换（太阳能电池）

### 第4节 能源与可持续发展
**核心知识点：**
1. 能量转化的方向性：有些能量转化是不可逆的
2. 节约能源，开发新能源
