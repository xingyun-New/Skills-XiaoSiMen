## 推荐内容格式

每个教材文件建议按以下结构组织：

```markdown
# X年级XX（上册 + 下册）

## 上册

### 第一单元：[单元标题]

#### 第1课 [课题名称]

**核心知识点：**
1. [知识点1]：[详细内容]
2. [知识点2]：[详细内容]

**重点概念：**
- [概念名称]：[定义/解释]

**易错点：**
- [常见错误]→[正确理解]

**记忆口诀：**
- [口诀内容]

---

### 第二单元：[单元标题]
...

## 下册

### 第一单元：[单元标题]
...
```

## 内容来源建议

1. **电子课本原文**：直接复制教材正文中的核心段落
2. **课后习题**：课本上的练习题和思考题
3. **教师用书要点**：教学重点、难点提示
4. **中考考纲**：当地中考考试大纲中的具体考点清单

## 注意事项

- 每个文件建议控制在合理长度内，过长的内容可拆分为多个文件
- 优先补充高频考点和易错知识点
- 如果教材版本不是人教版，请在文件开头注明版本
- 内容更新后，skill 会自动使用最新知识点出题
