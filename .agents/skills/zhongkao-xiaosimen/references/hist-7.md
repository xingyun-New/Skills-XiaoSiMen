# 七年级历史（上册 + 下册）

> 待补充：请参照 [textbook-guide.md](textbook-guide.md) 的格式添加人教版七年级历史教材知识点。

## 上册 知识框架（中国古代史：史前—魏晋南北朝）

### 第一单元 史前时期
- 元谋人、北京人
- 河姆渡文化、半坡文化
- 炎帝黄帝、尧舜禹

### 第二单元 夏商周时期
- 夏朝建立、商朝兴亡
- 西周分封制、宗法制
- 春秋战国、百家争鸣

### 第三单元 秦汉时期
- 秦统一六国、秦始皇
- 陈胜吴广起义
- 汉武帝大一统、丝绸之路

### 第四单元 三国两晋南北朝时期
- 三国鼎立
- 西晋统一与灭亡
- 北魏孝文帝改革

## 下册 知识框架（中国古代史：隋唐—明清）

### 第一单元 隋唐时期
- 隋朝统一、大运河、科举制
- 唐太宗与贞观之治
- 武则天、唐玄宗与开元盛世

### 第二单元 辽宋夏金元时期
- 北宋建立、重文轻武
- 经济重心南移
- 元朝统一、行省制度

### 第三单元 明清时期
- 明朝建立、朱元璋
- 郑和下西洋
- 清朝前期统治、闭关锁国
