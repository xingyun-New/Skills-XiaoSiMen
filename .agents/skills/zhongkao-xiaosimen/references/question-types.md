# 题型模板与示例

所有题型的标准格式、命题要求和完整示例。

## 一、选择题

### 格式要求
- 4个选项（A/B/C/D），有且仅有1个正确答案
- 题干表述清晰无歧义，避免否定式设问（如必须用否定式则加粗"不"字）
- 四个选项长度接近，语法结构一致
- 正确答案在A/B/C/D中均匀分布

### 模板

```markdown
### 第X题（选择题 | 难度：★★☆）

**【题目】** [题干内容]

A. [选项A]
B. [选项B]
C. [选项C]
D. [选项D]

**【提示】**
- 💡 提示1（方向）：[指出本题考查的知识范围]
- 📖 提示2（知识）：[给出关键知识点内容]
- 🎯 提示3（思路）：[给出解题思路，接近答案]

**【答案】** [正确选项字母]

**【知识点卡片】**
- 📚 考点：[具体知识点名称]
- 📖 出处：人教版X年级X册 第X单元/第X课
- 💡 关键记忆：[一句话核心记忆]
- ⚠️ 易错提醒：[说明为什么A/B/C是干扰项，学生容易犯什么错]
- 🔗 关联知识：[相关的其他知识点]
```

### 提示设计原则

| 级别 | 内容要求 | 扣分 |
|------|---------|------|
| 提示1（方向） | 只告诉考哪个章节/知识点方向，不透露具体内容 | -30% |
| 提示2（知识） | 给出解题需要的核心知识点，但不直接指向答案 | -60% |
| 提示3（思路） | 给出排除法或推理过程，接近给出答案 | -90% |

### 示例（地理）

```markdown
### 第1题（选择题 | 难度：★★☆）

**【题目】** 下列山脉中，属于我国地势第一级阶梯与第二级阶梯分界线的是：

A. 大兴安岭
B. 昆仑山脉
C. 太行山脉
D. 雪峰山

**【答案】** B

**【知识点卡片】**
- 📚 考点：中国地势三级阶梯的分界线
- 📖 出处：人教版八年级上册 第二章 中国的自然环境
- 💡 关键记忆：一二阶梯分界线——昆仑山、祁连山、横断山脉；二三阶梯分界线——大兴安岭、太行山、巫山、雪峰山
- ⚠️ 易错提醒：大兴安岭、太行山、雪峰山都是分界线，但它们是二三阶梯的分界线，不是一二阶梯的
- 🔗 关联知识：中国地势西高东低呈阶梯状分布的特征及其对气候、河流的影响
```

## 二、判断题

### 格式要求
- 陈述句形式，表述完整
- 错误题目要有明确的可辨别错误点（不能模棱两可）
- 正确和错误题目各占约50%

### 模板

```markdown
### 第X题（判断题 | 难度：★☆☆）

**【题目】** [陈述句]（  ）

**【答案】** ✓正确 / ✗错误

**【知识点卡片】**
- 📚 考点：[知识点]
- 📖 出处：[教材位置]
- 💡 关键记忆：[正确表述]
- ⚠️ 易错提醒：[如果是错误题，说明正确的应该是什么]
```

### 示例（历史）

```markdown
### 第3题（判断题 | 难度：★☆☆）

**【题目】** 辛亥革命结束了中国两千多年的封建制度。（  ）

**【答案】** ✗错误

**【知识点卡片】**
- 📚 考点：辛亥革命的历史意义
- 📖 出处：人教版八年级上册 第三单元 第9课
- 💡 关键记忆：辛亥革命结束的是"封建帝制"（君主专制制度），不是"封建制度"。封建制度的彻底废除是在新中国成立后的土地改革完成。
- ⚠️ 易错提醒："封建帝制"≠"封建制度"，这是中考高频易错点
```

## 三、填空题

### 格式要求
- 每空只有唯一标准答案（或已列明的等价答案）
- 空格用下划线表示，标注每空分值
- 避免一个句子中挖空过多导致无法理解句意

### 模板

```markdown
### 第X题（填空题 | 难度：★☆☆）

**【题目】** [含有______的陈述句]。

**【答案】** ① ______ ② ______

**【知识点卡片】**
- 📚 考点：[知识点]
- 📖 出处：[教材位置]
- 💡 关键记忆：[包含答案的完整句子]
```

### 示例（生物）

```markdown
### 第 5 题（填空题 | 难度：★☆☆）

**【题目】** 绿色植物通过光合作用，利用______能，把______和水转变成储存能量的有机物，并释放出氧气。

**【答案】** ① 光 ② 二氧化碳

**【知识点卡片】**
- 📚 考点：光合作用的概念
- 📖 出处：人教版七年级上册 第三单元 第四章
- 💡 关键记忆：光合作用公式——二氧化碳 + 水 →(光能/叶绿体) 有机物 + 氧气
- 🔗 关联知识：呼吸作用是光合作用的逆过程；光合作用场所是叶绿体
```

### 多空填空题格式（HTML 交互专用）

⚠️ **重要格式要求**：

当填空题有多个空时，必须使用**中文分号 `；`** 分隔答案，**不能**使用数组或英文逗号！

**正确格式示例：**

```json
{
  "type": "fillblank",
  "difficulty": 1,
  "question": "我国的基本政治制度包括：中国共产党领导的______和政治协商制度、______制度、基层群众自治制度。",
  "answer": ["多党合作；民族区域自治"],
  "hints": [
    "💡 方向提示：本题考查我国的基本政治制度",
    "📖 关键知识：基本政治制度有三个，根本政治制度只有一个",
    "🎯 解题思路：第一个空填「多党合作」，第二个空填「民族区域自治」"
  ],
  "knowledgeCard": {
    "topic": "我国的基本政治制度",
    "source": "人教版八年级下册 第三单元 第六课",
    "keyMemory": "基本政治制度有三个，根本政治制度只有一个（人民代表大会制度）",
    "commonMistake": "不要将「多党合作」写成「多党制」，我国不是多党制国家",
    "relatedTopics": "根本政治制度、人民代表大会制度"
  }
}
```

**错误格式示例（禁止使用）：**

```json
// ❌ 错误：不能使用数组分别存放每个空的答案
"answer": ["多党合作", "民族区域自治"]

// ❌ 错误：不能使用英文逗号分隔
"answer": ["多党合作，民族区域自治"]
```

**交互方式：**
- **单空题目**：显示一个输入框，学生填写后提交
- **多空题目**：显示多个独立输入框（标注"第 1 空"、"第 2 空"...），学生分别填写每个空
- **提交验证**：点击提交后，系统分别验证每个空的答案
- **即时反馈**：
  - 每个输入框会用颜色标记：绿色表示正确，红色表示错误
  - 显示"各空判分详情"，包括：
    - 每空的判断结果（✓正确/✗错误）
    - 学生填写的答案
    - 错误空的正确答案
  - 统计答对的空数（如"共 3 空，答对 2 空"）
- **计分规则**：多空题目必须全部答对才算正确，获得积分；否则不得分，计入错题

**注意事项：**
- `answer` 数组中第一个元素为用**中文分号 `；`** 分隔的标准答案
- 可提供的等价答案也放在 `answer` 数组中（同样用分号分隔）
- 题目中的空格数量必须与答案中的分号数量 +1 一致
- 多空题目会在题干中用 `____` 明确表示每个空的位置

## 四、材料分析题

### 格式要求
- 提供材料（文字/数据/图表描述），设 2-3 个小问
- 小问由浅入深：概括信息 → 分析原因 → 综合评价
- 材料来源标注（可为改编）

### 模板

```markdown
### 第 X 题（材料分析题 | 难度：★★★ | X 分）

**【材料】**
[材料内容，可含文字、数据表格、图表描述等]

**【设问】**
(1) [概括类问题]（X 分）
(2) [分析类问题]（X 分）
(3) [综合/评价类问题]（X 分）

**【参考答案】**
(1) [答案要点]
(2) [答案要点]
(3) [答案要点]

**【知识点卡片】**
- 📚 考点：[涉及的主要知识点]
- 📖 出处：[教材位置]
- 💡 答题技巧：[该题型的答题方法和注意事项]
- ⚠️ 易错提醒：[常见丢分点]
```

**交互流程说明：**
1. 学生阅读材料（紫色边框区域）
2. 在每道小题的文本框中输入答案
3. 点击「提交答案」按钮
4. 显示参考答案和评分标准
5. 学生自评：「完全正确」/「部分正确」/「错误」
6. 根据自评获得相应分数（100%/50%/0%）

**计分规则：**
- 完全正确：该题总分 × 100% + 连击奖励
- 部分正确：该题总分 × 50% + 连击奖励
- 错误：0 分，计入错题本
- 使用提示会扣分（30%/60%/90%）

### 示例（道法）

```markdown
### 第9题（材料分析题 | 难度：★★☆ | 6分）

**【材料】**
小明在网上购买了一双运动鞋，收到后发现是假冒伪劣产品，与商家协商退款被拒。小明的父亲建议他"算了，就当花钱买教训"，但小明认为应该维护自己的合法权益。

**【设问】**
(1) 商家的行为侵犯了小明的什么权利？（2分）
(2) 小明可以通过哪些途径维护自己的合法权益？（至少写出两种）（2分）
(3) 这件事给我们什么启示？（2分）

**【参考答案】**
(1) 商家侵犯了小明作为消费者的公平交易权和知情权（安全保障权亦可）。
(2) ①与商家协商和解；②请求消费者协会调解；③向市场监督管理部门投诉举报；④向人民法院提起诉讼。（任写两种即可）
(3) ①我们要增强法治意识和维权意识，当合法权益受到侵害时要勇于依法维权；②要学会运用法律武器保护自己的合法权益；③消费时要注意保留证据（如聊天记录、支付凭证等）。

**【知识点卡片】**
- 📚 考点：消费者权益保护、维权途径
- 📖 出处：人教版八年级上册 第二单元 遵守社会规则
- 💡 答题技巧：维权途径题按"协商→调解→投诉→诉讼"的顺序作答，由成本低到高排列
- ⚠️ 易错提醒：不要把"找媒体曝光"作为法定维权途径；"向法院起诉"不要写成"报警"（民事纠纷与刑事案件的区别）
```

## 五、连线/匹配题（补充题型）

### 模板

```markdown
### 第X题（匹配题 | 难度：★☆☆）

**【题目】** 将左侧内容与右侧对应项用线连接：

| 左栏 | | 右栏 |
|------|---|------|
| A. XX | | ① XX |
| B. XX | | ② XX |
| C. XX | | ③ XX |
| D. XX | | ④ XX |

**【答案】** A—③  B—①  C—④  D—②

**【知识点卡片】**
- 📚 考点：[知识点]
- 💡 关键记忆：[记忆技巧]
```

## 六、命题质量自检清单

每组题目生成后，按此清单自检：

- [ ] 知识点是否准确（对照教材）
- [ ] 题干表述是否清晰无歧义
- [ ] 选择题干扰项是否有合理迷惑性
- [ ] 正确答案分布是否均匀（A/B/C/D各约25%）
- [ ] 难度分布是否符合要求
- [ ] 每题是否都附有完整的知识点卡片
- [ ] 知识点是否覆盖了不同考点（避免重复）
- [ ] 材料分析题的设问是否由浅入深
- [ ] 答案是否完整、准确
- [ ] 易错提醒是否切中学生真实痛点
