# 八年级生物（上册 + 下册）

> 待补充：请参照 [textbook-guide.md](textbook-guide.md) 的格式添加人教版八年级生物教材知识点。

## 上册 知识框架

### 第五单元 生物圈中的其他生物
- 动物的主要类群（腔肠动物、扁形动物、线形动物、环节动物、软体动物、节肢动物）
- 鱼类、两栖动物、爬行动物
- 鸟类、哺乳动物
- 动物的运动（骨、关节、肌肉的协调配合）
- 动物的行为（先天性行为、学习行为）
- 动物在生物圈中的作用

### 第六单元 生物的多样性及其保护
- 细菌和真菌
- 细菌（结构特征、繁殖方式）
- 真菌（酵母菌、霉菌、蘑菇）
- 细菌和真菌在自然界中的作用
- 人类对细菌和真菌的利用
- 病毒

## 下册 知识框架

### 第七单元 生物的生殖
- 植物的生殖（有性生殖、无性生殖）
- 昆虫的生殖和发育（完全变态、不完全变态）
- 两栖动物的生殖和发育
- 鸟的生殖和发育

### 第八单元 生物的遗传与变异
- 基因控制生物的性状
- 基因与染色体（DNA、基因、染色体的关系）
- 基因的显性和隐性
- 人的性别遗传
- 生物的变异

### 第九单元 生物的进化
- 地球上生命的起源
- 生物进化的历程
- 生物进化的原因
- 人类的起源

### 第十单元 生物多样性及其保护
- 进化与生物多样性
- 人与自然和谐共生
