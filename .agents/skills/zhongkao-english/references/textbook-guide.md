# 教材内容添加指南

本 skill 的出题精度取决于教材知识点的完整程度。请按以下方式补充各科教材内容。

## 文件对应关系

| 文件 | 对应教材 |
|------|---------|
| `english-7.md` | 外研版（New Standard English）七年级英语（上册 + 下册） |
| `english-8.md` | 外研版（New Standard English）八年级英语（上册 + 下册） |
| `english-9.md` | 外研版（New Standard English）九年级英语（上册 + 下册） |

## 推荐内容格式

每个教材文件建议按以下结构组织：

```markdown
# X年级英语（上册 + 下册）

## 上册

### Module 1: [模块标题]

**核心语法：**
- [语法点]：[规则说明]
- [结构公式]

**重点句型：**
- [句型1]
- [句型2]

**核心词汇：**
- [词汇列表，含词性和中文释义]

**易错点：**
- [常见错误]→[正确用法]

**话题写作：**
- [本模块话题的写作模板或句型]

---

### Module 2: [模块标题]
...
```

## 重点内容优先级

### 必须优先补充的内容

1. **语法规则**：每个模块的核心语法点、结构公式、使用规则
2. **重点句型**：每模块的核心句型及中文释义
3. **易混辨析**：高频易混语法点的对比表格
4. **不规则动词表**：完整的不规则动词过去式和过去分词

### 建议补充的内容

5. 核心词汇表（含音标、词性、中文释义）
6. 话题写作模板和范文
7. 中考真题高频考点统计

## 内容来源建议

1. **教材原文**：每模块的 Language Practice 和 Grammar 部分
2. **课后练习**：教材上的 Workbook 和 Around the World 部分
3. **教师用书**：教学重点、难点、教学建议
4. **中考考纲**：当地中考词汇表和语法考查范围

## 注意事项

- 每个文件建议控制在合理长度内
- 优先补充中考高频考点
- 语法说明建议用中文，例句用英文
- 词汇建议按模块或话题分组，便于出题时定位
- 外研版以 Module（模块）为单位组织，每个 Module 包含 3 个 Unit
- 内容更新后，skill 会自动使用最新知识点出题
