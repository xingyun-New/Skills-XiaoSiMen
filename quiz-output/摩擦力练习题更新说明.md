## 更新内容

### ✅ 问题 1：添加提交答案按钮

**问题描述**：
- 之前版本：学生选择答案后直接禁用按钮，没有立即显示反馈，必须点击"下一题"才能看到结果
- 导致学生无法及时知道自己是否答对，影响学习效果

**修复方案**：
1. 新增"提交答案"按钮，位于提示框下方
2. 答题流程优化为：
   - 学生选择答案 → 点击"提交答案" → 立即显示对错和解析
   - 然后点击"下一题"继续
3. 提交后立即显示：
   - ✅ 绿色标记正确答案
   - ❌ 红色标记错误答案
   - 📝 详细解析
   - 📚 知识点卡片

**代码改动**：
- 添加 `submitAnswer()` 函数处理答案提交
- 修改 `selectOption()` 函数启用提交按钮
- 优化 `nextQuestion()` 函数移除自动延迟跳转

---

### ✅ 问题 2：实现三级提示系统

**问题描述**：
- 之前版本：点击提示直接显示全部提示内容
- 无法引导学生逐步思考，提示过于直接

**修复方案**：
1. 将每题的 `hint` 字段从字符串改为数组（3 个提示）
   ```javascript
   hint: [
     "💡 提示 1：A 和 B 都静止，它们之间有没有相对运动趋势？",
     "💡 提示 2：静摩擦力产生的条件是什么？",
     "💡 提示 3：A 只受到重力和支持力，水平方向没有其他力"
   ]
   ```

2. 修改 `toggleHint()` 函数实现三级渐进提示：
   - 第 1 次点击：显示提示 1（最浅层提示）
   - 第 2 次点击：显示提示 2（中等深度提示）
   - 第 3 次点击：显示提示 3（最详细提示）
   - 显示计数器：（1/3）、（2/3）、（3/3）

3. 提示设计原则：
   - **提示 1**：启发式问题，引导学生思考关键条件
   - **提示 2**：回顾相关知识点或公式
   - **提示 3**：给出具体解题思路或计算步骤

**代码改动**：
- 添加 `hintLevel` 变量追踪当前提示级别
- 修改 `toggleHint()` 函数支持数组和三级显示
- 更新所有 10 道题的 `hint` 字段为三级提示数组

**提示示例（第 1 题）**：

- **提示 2**：静摩擦力产生的条件是什么？

### 📊 界面优化

- **提交答案按钮**：蓝色渐变，选择答案后启用

- **知识点卡片**：蓝色边框，包含核心知识点、解题技巧、易错提醒

## 技术实现

### 核心函数

1. **`submitAnswer()`**：处理所有题型的答案提交
   - 选择题/判断题：检查选中选项
   - 填空题：调用 `checkFillAnswer()`
   - 计算题：调用 `showCalculationAnswer()`

2. **`toggleHint(el)`**：三级提示切换
   ```javascript
   hintLevel++;
   if (hintLevel > 3) hintLevel = 3;
   el.textContent = q.hint[hintLevel - 1] + `（${hintLevel}/3）`;
   ```

3. **`nextQuestion()`**：跳转下一题
   - 重置提示级别 `hintLevel = 0`
   - 重置选择状态
   - 禁用提交按钮，启用下一题按钮
