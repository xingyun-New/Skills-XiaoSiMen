💡 知识点

k (比例系数): 决定双曲线的位置和形状

  • k > 0: 图像在第一、三象限
  • k < 0: 图像在第二、四象限
  • |k| 越大，图像离坐标轴越远
  • k ≠ 0 (k = 0 时不是反比例函数)
y = 1/x

📚 学习模式

反比例函数 y = k/x

反比例函数 是形如 y = k/x 的函数，其中:

  • k 是比例系数 (k ≠ 0)
  • x 是自变量，y 是因变量
  • 图像是双曲线，有两个分支

图像特征

  • 双曲线关于原点对称
  • 与坐标轴没有交点
  • x ≠ 0, y ≠ 0

反比例函数的特殊性质

1️⃣ 几何意义

对于图像上任意一点 P(x, y)，过 P 作 x 轴、y 轴的垂线，所得矩形面积为 |k|

面积 = |x| × |y| = |xy| = |k|

2️⃣ 对称性

反比例函数图像关于原点中心对称

也关于直线 y = x 和 y = -x 对称

3️⃣ 增减性

  • k > 0: 在每个象限内，y 随 x 增大而减小
  • k < 0: 在每个象限内，y 随 x 增大而增大

4️⃣ 与正比例函数的交点

联立 y = k₁x 和 y = k₂/x，解得交点关于原点对称
